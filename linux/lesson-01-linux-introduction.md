# Linux Introduction

## What is Linux?
Linux is an open-source operating system used in servers, computers, cybersecurity, and many other systems.

## Why cybersecurity professionals learn Linux
- Many servers run Linux.
- Security tools are commonly built for Linux.
- Linux gives powerful control through the terminal.
